# Cep
[![NPM](https://img.shields.io/npm/l/react)](https://github.com/PedroRoca7/Cep/blob/main/LICENSE) 

# Sobre o projeto

Cep é uma aplicativo mobile que utiiza a API da ViaCep para trazer informações apenas digitando o cep do local.

O Cep funciona da seguinte maneira o usuário digita o numero do cep que ele deseja, e o app apresenta o nome da rua o bairro e o DDD e caso o usuário deseje ver imagens do local com apenas um click o app redireciona ele para o google imagens já com as imagens do local.
